JS taak: eventhandlers

Opdracht:

  • Laat verschillende berichtjes verschijnen op een pagina, niet noodzakelijk in deze volgorde:
    • bij het klikken op een hyperlink, verhinder hier het doorlinken
    • door het klikken op een knop
    • bij een mouseover van een image
    • bij het klikken op een submitknop van een formulier, verhinder hier de submit
  • • Het bericht bevat telkens dezelfde info, probeer hiervoor dezelfde functie te gebruiken:
    • het type event
    • de nodeName en de id van het target object
    • als het element een src attribuut heeft, toon het
    • als het element een href attribuut heeft, toon het
    • als het element een action attribuut heeft, toon het
    • als het element een method attribuut heeft, toon het
  • de knop "stop alle ingestelde eventhandlers" verbreekt het vorige
  • probeer unobtrusive te werken
